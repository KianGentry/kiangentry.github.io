Features - Release 11

Game Engine

Generic 2D Game Engine: EYN-OS now includes a powerful game engine that supports human-readable text file formats. Create and play multiple game types including Snake, Maze Runner, and custom games with configurable objects, entities, and AI.

  • Human-readable text file format for easy game creation
  • Multiple game types: Snake, Maze, Generic
  • Collision detection and combat systems
  • Simple AI for enemies and NPCs
  • Health and damage systems
  • Collectible items and objectives
  • Arrow key and WASD support
  • Level-based games with custom layouts

Development Tools

Built-in Assembler: EYN-OS includes a NASM-compatible assembler that allows you to write and compile assembly code directly within the operating system.

  • NASM-compatible syntax
  • Support for basic x86 instructions (mov, add, sub, jmp, call, ret, int)
  • Label resolution and forward references
  • Data directives (db, dw, dd)
  • Generates EYN executable format
  • Complete development environment without external tools

Math Utilities

Advanced Mathematical Functions: Comprehensive math library including random number generation, sorting algorithms, and search functions.

  • Linear Congruential Generator for random numbers
  • Quicksort implementation for string arrays
  • Boyer-Moore string search algorithm
  • Range-based random number generation
  • Efficient memory usage for low-end systems

Enhanced Filesystem

EYNFS with Advanced Operations: Improved filesystem with better file handling and operations.

  • Custom EYNFS filesystem with improved performance
  • Advanced file operations (copy, move, delete)
  • Directory management and traversal
  • Support for large files (fixed truncation issues)
  • FAT32 compatibility for existing filesystems

User Interface

Enhanced Shell and TUI: Improved user interface with better command handling and text user interface system.

  • Command history with arrow key navigation
  • Interactive help system
  • Enhanced text editor with improved file handling
  • Consistent TUI framework for all applications
  • Color support and keyboard handling

System Features

  • Custom Filesystem (EYNFS): EYN-OS uses its own filesystem, designed for simplicity and efficiency. Includes a custom formatting tool and supports advanced file operations.
  • Built-in Text Editor: EYN-OS features an enhanced text editor accessible from the shell, supporting basic editing, file loading, and saving with improved file size handling.
  • Enhanced Shell: The OS includes an improved shell with familiar commands, command history, and comprehensive help system.
  • Runs on Real Hardware and QEMU: Bootable via QEMU or from a USB stick on real Intel i386 hardware.
