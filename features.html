Features

  • Custom Filesystem (EYNFS): EYN-OS uses its own filesystem, designed for simplicity and efficiency. Includes a custom formatting tool and supports basic file operations.
  • Custom C Compiler: A port/adaptation of TinyCC is included, allowing for C development directly within or for EYN-OS. (Work in progress)
  • Built-in Text Editor: EYN-OS features a text editor accessible from the shell, supporting basic editing, file loading, and saving.
  • Basic Shell: The OS includes a shell with familiar commands (help, ver, etc.), making it easy to interact with the system.
  • Runs on Real Hardware and QEMU: Bootable via QEMU or from a USB stick on real Intel i386 hardware.
