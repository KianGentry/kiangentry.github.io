Features - Release 13

Game Engine

Generic 2D Game Engine: EYN-OS now includes a powerful game engine that supports human-readable text file formats. Create and play multiple game types including Snake, Maze Runner, and custom games with configurable objects, entities, and AI.

The game engine has been completely rewritten to support a generic 2D engine that can interpret human-readable text files. This replaces the old binary format with a more flexible and user-friendly approach, making game creation accessible to users without programming experience.

  • Human-readable text file format for easy game creation
  • Multiple game types: Snake, Maze, Generic
  • Collision detection and combat systems
  • Simple AI for enemies and NPCs
  • Health and damage systems
  • Collectible items and objectives
  • Arrow key and WASD support
  • Level-based games with custom layouts

Complete Documentation: Game Engine Documentation (GitHub) | Detailed Web Documentation

Development Tools

Built-in Assembler: EYN-OS includes a NASM-compatible assembler that allows you to write and compile assembly code directly within the operating system.

The built-in assembler provides a complete development environment without external tools. It supports the full x86 instruction set and generates EYN executable format files that can be run directly within the operating system.

  • NASM-compatible syntax
  • Support for basic x86 instructions (mov, add, sub, jmp, call, ret, int)
  • Label resolution and forward references
  • Data directives (db, dw, dd)
  • Generates EYN executable format
  • Complete development environment without external tools

Complete Documentation: Assembler Documentation (GitHub) | Detailed Web Documentation

Math Utilities

Advanced Mathematical Functions: Comprehensive math library including random number generation, sorting algorithms, and search functions.

The math utilities provide essential algorithms and functions for development and system operations. These include efficient random number generation using a Linear Congruential Generator, fast sorting with Quicksort, and advanced search capabilities with the Boyer-Moore algorithm.

  • Linear Congruential Generator for random numbers
  • Quicksort implementation for string arrays
  • Boyer-Moore string search algorithm
  • Range-based random number generation
  • Efficient memory usage for low-end systems

Complete Documentation: Math Utilities Documentation (GitHub)

Enhanced Filesystem

EYNFS with Advanced Operations: Improved filesystem with better file handling and operations.

EYN-OS features a custom filesystem (EYNFS) designed for simplicity and efficiency, along with FAT32 compatibility for existing filesystems. The filesystem layer provides comprehensive file operations and directory management with improved performance and reliability.

  • Custom EYNFS filesystem with improved performance
  • Advanced file operations (copy, move, delete)
  • Directory management and traversal
  • Support for large files (fixed truncation issues)
  • FAT32 compatibility for existing filesystems

Complete Documentation: EYNFS Documentation (GitHub) | FAT32 Support (GitHub)

User Interface

Enhanced Shell and TUI: Improved user interface with better command handling and text user interface system.

The user interface system provides a comprehensive command-line environment with a text user interface framework. This includes an enhanced shell with command history, interactive help system, and consistent TUI components for all applications.

  • Command history with arrow key navigation
  • Interactive help system
  • Enhanced text editor with improved file handling
  • Consistent TUI framework for all applications
  • Color support and keyboard handling

Complete Documentation: Shell System (GitHub) | TUI Framework (GitHub) | Help System (GitHub)

System Features

  • Custom Filesystem (EYNFS): EYN-OS uses its own filesystem, designed for simplicity and efficiency. Includes a custom formatting tool and supports advanced file operations.
  • Built-in Text Editor: EYN-OS features an enhanced text editor accessible from the shell, supporting basic editing, file loading, and saving with improved file size handling.
  • Enhanced Shell: The OS includes an improved shell with familiar commands, command history, and comprehensive help system.
  • Runs on Real Hardware and QEMU: Bootable via QEMU or from a USB stick on real Intel i386 hardware.

Advanced Memory Management

Predictive Memory System: EYN-OS includes an advanced memory management system that predicts memory usage patterns and optimizes allocation strategies for better performance.

The predictive memory system analyzes usage patterns to optimize memory allocation, reducing fragmentation and improving overall system performance. It includes zero-copy operations and efficient allocation strategies designed for low-end systems.

  • Memory usage pattern analysis and prediction
  • Zero-copy memory operations for better performance
  • Efficient allocation strategies to reduce fragmentation
  • Memory leak detection and prevention
  • Performance monitoring and optimization

Complete Documentation: Predictive Memory System (GitHub)
