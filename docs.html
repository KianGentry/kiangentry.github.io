Assembler Example

Here's a simple "Hello, World!" assembly program:

; Simple "Hello, World!" program
section .text
global _start

_start:
    mov eax, 4          ; syscall number for write
    mov ebx, 1          ; file descriptor (stdout)
    mov ecx, message    ; message to write
    mov edx, 13         ; message length
    int 0x80            ; make syscall
    
    mov eax, 1          ; syscall number for exit
    mov ebx, 0          ; exit code
    int 0x80            ; make syscall

section .data
message:
    db "Hello, World!", 10  ; 10 is newline

Filesystem Commands

Common filesystem operations in EYN-OS:

ls              # List files and directories
cd         # Change directory
cd ..           # Go to parent directory
cd /            # Go to root directory
clear           # Clear screen

# File operations
read filename.txt    # Display file contents
write newfile.txt   # Create and edit file
copy source.txt dest.txt  # Copy file
move file.txt /backup/    # Move file to directory
del filename.txt     # Delete file
deldir directory     # Delete directory
makedir new_dir      # Create directory

# Development
assemble test.asm test.eyn  # Assemble code
run program.eyn        # Execute program
game snake             # Launch game
