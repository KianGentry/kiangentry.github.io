Documentation

Overview

EYN-OS is a 32-bit operating system for Intel i386 hardware and QEMU. It features a custom filesystem (EYNFS), a basic shell, a built-in text editor (Write), and a work-in-progress C compiler. EYN-OS is designed for learning, experimentation, and extensibility.

Shell Commands

The EYN-OS shell provides a familiar command-line interface. Some available commands include:

  • help — List available commands
  • ver — Show OS version
  • ls — List files in the current directory
  • cd — Change directory
  • cat — View file contents
  • write <filename> — Open the Write text editor
  • format — Format a disk with EYNFS
  • mount — Mount a filesystem
  • run — Run a program
  • exit — Exit the shell

Write (Text Editor)

Write is the built-in text editor for EYN-OS. It allows you to create and edit text files directly within the OS. Features include:

  • Basic text editing (insert, delete, navigate)
  • Open and save files to EYNFS
  • Keyboard shortcuts: Ctrl+O to save, Ctrl+X to exit

To launch Write, use the shell command: write <filename>

EYNFS (Filesystem)

EYNFS is the custom filesystem used by EYN-OS. It is designed for simplicity and efficiency, supporting basic file and directory operations. You can format a disk image with EYNFS using the eynfs_format tool, and interact with files using shell commands.

  • Format a disk: format (from the shell) or eynfs_format (external tool)
  • Mount a filesystem: mount
  • File operations: ls, cd, cat, write, etc.
