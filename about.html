About EYN-OS Release 11

EYN-OS is a custom 32-bit operating system project by Kian Gentry. It is open source and designed for learning, experimentation, and providing a foundation for further OS development. EYN-OS Release 11 represents a major milestone with significant new features and improvements.

Release 11 Highlights

This release introduces several major new features that transform EYN-OS from a basic operating system into a comprehensive development and gaming platform:

  • Game Engine: A complete 2D game engine with human-readable text files, multiple game types, and AI systems
  • Development Tools: Built-in NASM-compatible assembler for native development
  • Math Utilities: Advanced mathematical functions including random generation, sorting, and search algorithms
  • Enhanced Filesystem: Improved EYNFS with better file operations and large file support
  • Documentation: Comprehensive documentation system covering all aspects of the OS

Development Philosophy

EYN-OS follows the philosophy of "reinventing the wheel" - building everything from scratch to understand and control every aspect of the system. This approach provides:

  • Educational Value: Clear, understandable code for learning purposes
  • Full Control: Complete control over system behavior and performance
  • Simplicity: No unnecessary complexity or external dependencies
  • Customization: Tailored to specific needs and requirements

Technical Specifications

  • Architecture: Intel x86 (32-bit)
  • Bootloader: GRUB (Multiboot 1.0 compliant)
  • Memory Model: Flat memory model (no segmentation/paging)
  • Target Hardware: Low-end systems (below 16MB RAM)
  • ISO Size: 6.3MB (78% reduction from original 29MB)
  • Filesystem: Custom EYNFS with FAT32 compatibility
