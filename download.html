Download EYN-OS Release 11

Release 11 - Major Update: EYN-OS has received a massive update with new features including a generic 2D game engine, built-in assembler, advanced math utilities, and enhanced filesystem operations.

Download Files

  • EYNOS.iso (bootable image - 6.3MB)
  • eynfs.img (filesystem image)

What's New in Release 11

  • Game Engine: Generic 2D game engine with human-readable text files, multiple game types (Snake, Maze, Generic), collision detection, AI, and health systems
  • Assembler: Built-in NASM-compatible assembler for development
  • Math Utilities: Random number generation, quicksort, Boyer-Moore search algorithm
  • Enhanced Filesystem: Improved EYNFS with better file operations and large file support
  • UI Improvements: Enhanced shell with command history, better TUI system
  • Documentation: Comprehensive documentation system included

How to Run

With QEMU:

qemu-system-i386 -cdrom EYNOS.iso -hda tmp/boot/disk.img -hdb tmp/boot/eynfs.img -boot d

Or simply run:

make run

On Real Hardware: Flash the ISO to a USB drive and boot on compatible Intel i386 hardware.

System Requirements

  • Architecture: Intel x86 (32-bit)
  • Minimum RAM: 1MB
  • Recommended RAM: 4MB+
  • Storage: Any size (EYNFS formatted)
  • Boot Method: GRUB (Multiboot 1.0 compliant)
