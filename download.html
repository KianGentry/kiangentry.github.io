Download EYN-OS

Development Build: EYN-OS is under active development. Download the latest ISO and filesystem images below.

  • EYNOS.iso (bootable image)
  • eynfs.img (filesystem image)

How to Run

With QEMU:

qemu-system-i386 -cdrom EYNOS.iso -hda tmp/boot/disk.img -hdb tmp/boot/eynfs.img -boot d

Or simply run:

make run

On Real Hardware: Flash the ISO to a USB drive and boot on compatible Intel i386 hardware.
