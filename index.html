What is EYN-OS?

EYN-OS is a bare bones, open source operating system designed for learning, experimentation, and extensibility. It runs on real hardware and virtual machines, and includes its own filesystem (EYNFS), built-in assembler, game engine, and comprehensive development tools.

  • Custom EYNFS filesystem with advanced file operations
  • Built-in NASM-compatible assembler for development
  • Generic 2D game engine with multiple game types
  • Advanced math utilities (random generation, sorting, search)
  • Enhanced shell with command history and help system
  • Built-in text editor with improved file handling
  • Runs on QEMU and real Intel i386 hardware
