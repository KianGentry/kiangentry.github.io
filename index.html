Get EYN-OS

What is EYN-OS?

EYN-OS is a bare bones, open source operating system designed for learning, experimentation, and extensibility. It runs on real hardware and virtual machines, and includes its own filesystem (EYNFS), a work-in-progress C compiler, and a built-in text editor.

  • Custom EYNFS filesystem
  • Basic shell with familiar commands
  • Built-in text editor
  • Custom C compiler (in progress)
  • Runs on QEMU and real Intel i386 hardware
